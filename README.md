# Incogine
A leafstudiosDot engine. Now open source!

## Dependencies
- CMake
- C++17
- SDL2
- SDL2_image
- SDL2_ttf

## Build
> Recommended using Visual Studio Code with CMake Tools extension.

For Terminal compilers, run the following commands in the project root directory:
```
$ mkdir build
$ cd build
$ cmake ..
$ make
```
